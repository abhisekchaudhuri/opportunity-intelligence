# Opportunity Intelligence (OI)

Opportunity Intelligence (OI) is a disciplined operating system for identifying, evaluating, and pursuing high-leverage career opportunities.

OI turns a job search into an evidence-led decision process. It organizes company, people, market, talent, job, and signal intelligence; applies a transparent scoring framework; and preserves a durable record of decisions and assumptions over time.

## What this repository contains

- **Model** — the OI operating model, architecture, mandate, scoring framework, and milestones.
- **Knowledge** — structured research domains for opportunities and the surrounding market.
- **Analysis** — dated assessments and decision records.
- **Snapshots** — point-in-time captures of the opportunity landscape.
- **Templates** — reusable formats for research, scoring, and outreach.

## Guiding principles

1. Be selective: prioritize opportunities with asymmetric upside.
2. Use evidence: distinguish facts, signals, and hypotheses.
3. Make decisions explicit: document criteria, trade-offs, and confidence.
4. Learn continuously: update views as new evidence arrives.

## Status

The repository is initialized with the core OI structure. The model documents define the initial working system; knowledge and analysis directories are ready for research artifacts.
