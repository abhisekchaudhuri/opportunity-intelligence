# Architecture

OI is organized into five layers:

1. **Knowledge** — facts, sources, and observations.
2. **Signals** — changes that may alter an opportunity's value or timing.
3. **Analysis** — dated synthesis, hypotheses, and recommendations.
4. **Scoring** — a comparable assessment of opportunity quality.
5. **Action** — outreach, applications, relationship-building, and decisions.

Each layer should retain links to supporting evidence and state its confidence level.
